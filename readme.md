# React native - Expo WebView Sample Kit

    Converting WebApps to Mobile Apps using WebView (React Natvie , Expo )
  
## Get Started

Install the repository locally:

```bash
git clone https://github.com/akashsairaja/react-native-webview.git
```

## Installing Dependencies Locally

```bash
cd react-native-web-view
cmd npm install
```

### Run App Locally

```bash
expo start
```
